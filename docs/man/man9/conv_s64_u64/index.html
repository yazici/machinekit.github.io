HAL Component — CONV_S64_U64

NAME

conv_s64_u64 — Convert a value from s64 to u64

SYNOPSIS

conv_s64_u64

USAGE SYNOPSIS

loadrt conv_s64_u64 [count=N|names=name1[,name2…​]]

FUNCTIONS

conv_s64_u64.N

Update 'out' based on 'in'

PINS

in s64 in

out u64 out

out-of-range bit out - TRUE when 'in' is not in the range of u64

PARAMETERS

clamp bit rw - If TRUE, then clamp to the range of u64. If FALSE, then allow the value to "wrap around".

LICENCE

GPL
